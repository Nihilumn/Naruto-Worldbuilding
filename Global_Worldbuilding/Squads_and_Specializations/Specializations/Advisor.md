# Advisor

## Description
The Brain. The Advisor provides historical context, cultural knowledge, and strategic foresight to a Diplomatic Leader.

## Operational Variants
*   **Political Advisor:** Focuses on *Court Intrigue*. Knowing who holds power in a Daimyo's court.
*   **Military Advisor:** Focuses on *Force Assessment*. Determining if a potential ally is actually strong enough to be useful.
